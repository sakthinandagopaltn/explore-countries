# explore-countries
This repository is created for Python Advanced Pre-Work course with Code The Dream 

The API that is used is 

Option 1: Countries (countries.dev)
A database of every country in the world, with data on population, area, region, languages, capital city, and more.

Base endpoint: https://countries.dev/countries

Docs: https://countries.dev/docs

The project accesses this API and help you with selecting the right mode of transportation and the currency you need to have ,when travelling from source to destination!
